# Screenshot Plan

## Current Status

No screenshot assets are committed yet. Do not claim screenshots exist until they are captured from an emulator or physical Android device.

## Recommended Device

- Pixel 7 or Pixel 8 emulator.
- Android API 35.
- Light theme.
- Fresh app install.

## Demo Data

Before capturing screenshots:

1. Open Profile.
2. Tap `Load Demo Data`.
3. Confirm.
4. Return to Dashboard.

## Screenshot Checklist

Capture these screens:

- Welcome screen.
- Category selection.
- Business setup form.
- Setup review.
- Dashboard command center.
- Financial tracking section.
- Weekly Plan.
- Content Planner generated ideas.
- Content Calendar section.
- Retrospective.
- Business Report KPI section.
- Business Report export-ready text.
- Profile with Demo Mode controls.

## Quality Checks

- Text is readable.
- No clipped buttons.
- No overlapping cards.
- Demo data is visible.
- Safe disclaimers are visible where relevant.
- Status labels are visible, not only colors.
